1. SYSTEM ARCHITECTURE & BLOCK LEVEL OVERVIEW
-------------------------------------------------------------------------------
NeoRAG Space is engineered to parse, index, search, and answer complex queries 
from localized unstructured data stores completely offline. The data pipeline 
operates across six fully decoupled, asynchronous transaction boundaries:

Phase 1: Ingestion Engine [src/ingestor.py]
  - Multi-threaded file streams extract character strings page-by-page.
  - Native support for binary PDFs, structured transcript JSON arrays, and text volumes.
  - Generates immutable metadata tags to ensure downstream trace accuracy.

Phase 2: Semantic Sliding Chunker [src/chunker.py]
  - Divides continuous documents arrays into discrete 500-character tokens windows.
  - Enforces a 15% sliding character balance overlap window boundaries.
  - Prevents semantic loss or contextual cliff drops at technical data edge regions.

Phase 3: Transformer Embedder [src/embedder.py]
  - Leverages local Hugging Face 'all-MiniLM-L6-v2' SentenceTransformer models weights.
  - Maps alphanumeric token chunks into dense 384-dimensional floating spatial fields.
  - Operates totally offline inside native execution CPU registers with zero web data leaks.

Phase 4: Persistent Index Database [src/vector_store.py]
  - Uses binary object graph serialization to write arrays directly onto local filesystems.
  - Locks and compresses the vectorized knowledge bank inside 'vault_manager/vector_index.bin'.
  - Accommodates 15,860 deep technical structural nodes with high lookup capacity.

Phase 5: Matrix Similarity Arithmetic Core [src/vector_store.py]
  - Utilizes low-level optimized NumPy matrix operations to compute vector spatial indices.
  - Operates via Vector Angular Cosine Similarity Mathematics:
    Similarity Score = (Vector Q • Vector V) / (||Vector Q|| x ||Vector V||)
  - Filters and bubbles up the Top-K (K=3) context segments matching semantic intents.

Phase 6: Multi-Threaded Graceful UI Interface Runtime [app.py]
  - Spawns an isolated Python asynchronous thread immediately on runtime startup.
  - Background task maps the heavy vector matrix array chunks directly into active RAM.
  - Renders a Glassmorphism Cyberpunk progress loader in the UI during bootstrapping.
  - Routes factual contextual templates downstream into Ollama's local engine ('llama3').
